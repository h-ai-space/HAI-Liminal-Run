# H/AI — Liminal Run

Version visuelle corrigée.

## Lancer le jeu

Ouvrir `index.html` dans un navigateur.

## Contrôles

- Flèches gauche/droite : bouger
- Espace : interagir

## GitHub Pages

Mettre ces fichiers à la racine du repository :

- `index.html`
- `style.css`
- `game.js`
- dossier `assets`
- `LICENSE.txt`

Puis aller dans Settings → Pages → Deploy from branch → main → root.
