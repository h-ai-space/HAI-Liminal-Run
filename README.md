## Déploiement GitHub Pages

1. Supprime les anciens fichiers du repository.
2. Upload seulement :
   - `index.html`
   - `LICENSE.txt`
3. Va dans Settings → Pages.
4. Source : Deploy from a branch.
5. Branch : main.
6. Folder : /root.
7. Save.

Puis ouvre l'URL GitHub Pages.
